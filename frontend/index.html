How this UI maps to options
URL: paste a supported link. The UI detects the platform and narrows the available outputs.
YouTube blocks: if you see "confirm you're not a bot", try disabling VPN/proxy (e.g. NordVPN) or switching exit node.
Output:
MP3: audio-only export at the selected bitrate.
FLAC: lossless audio (bigger files).
MP4: video download when the platform supports it.
Instagram items: for carousels, choose first/all/selection (indices like 1,3-4).
Downloads: files are written to ./downloads/ and shown as links after completion.
